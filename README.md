# SHEC
El SmartHome Energy Challenge (SHEC) es un proyecto que busca optimizar el consumo energético en hogares mediante aplicaciones tecnológicas avanzadas. Utiliza IoT e inteligencia artificial para proporcionar monitoreo y control inteligente de dispositivos, promoviendo eficiencia, reducción de costos y sostenibilidad.
